# PRODIGY_DS_03

## Task 03 - Decision Tree Classifier

### Objective
Build a Decision Tree Classifier to predict whether a customer will purchase a product or service based on demographic and behavioral data.

### Dataset
Bank Marketing Dataset (UCI Machine Learning Repository)

### Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

### Steps Performed
1. Data Preprocessing
2. Label Encoding of Categorical Features
3. Train-Test Split
4. Decision Tree Model Training
5. Model Evaluation
6. Decision Tree Visualization

### Output
- Accuracy Score
- Decision Tree Visualization (decision_tree.png)

### Result
A Decision Tree Classifier was trained to predict customer subscription behavior based on demographic and marketing data.
